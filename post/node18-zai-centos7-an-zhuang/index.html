node18在Centos7安装

2025-01-09 1 min read # Linux

一开始不知道这个限制，捣鼓了很久，记录一下

从nodejs18开始就不支持Centos7了，这是因为centos7的gilbc版本比较低，因此需要安装非官方构建的版本。

三个方案来解决这个问题：

  • 从源代码安装 Node。这将允许您重新编译
  • 安装操作系统可支持的旧版本 Node
  • 将你的 Linux 操作系统升级到更新的版本。
